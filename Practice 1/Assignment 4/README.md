# Phân Tích Bố Cục và Các Phần Tử

## Cấu Trúc Chính

- **Form** được chia thành 3 phần:
    1. **Basic Info**
    2. **Contact Info**
    3. **Optional Info**
- Sử dụng bố cục **2 cột** (label - input).
- Có nhiều loại input khác nhau:
    - Text
    - Radio
    - Date
    - Dropdown
    - Textarea
    - Checkbox
    - CAPTCHA

## Các Phần Tử Cần Xây Dựng

- **Text inputs**: Trường nhập văn bản cơ bản.
- **Radio buttons**: Lựa chọn giới tính (Gender).
- **Date inputs**: Chọn ngày.
- **Dropdown/Select**: Danh sách lựa chọn.
- **Textarea với rich text editor**: Ô nhập văn bản nhiều dòng, hỗ trợ định dạng.
- **Checkboxes**: Lựa chọn nhiều giá trị.
- **CAPTCHA verification**: Xác thực người dùng.
- **Submit và Cancel buttons**: Nút gửi và hủy.

# Các Lệnh Cơ Bản của Git

| Lệnh                    | Công Dụng                                             |
| ----------------------- | ----------------------------------------------------- |
| `git init`              | Khởi tạo một repository Git mới                       |
| `git clone <url>`       | Sao chép một repository từ xa về máy local            |
| `git status`            | Kiểm tra trạng thái các file trong repository         |
| `git add <file>`        | Thêm file vào khu vực staging để chuẩn bị commit      |
| `git commit -m ""`      | Lưu các thay đổi đã staging với một thông điệp        |
| `git log`               | Xem lịch sử các commit                                |
| `git diff`              | So sánh sự khác biệt giữa các phiên bản file          |
| `git branch`            | Quản lý các nhánh (branch) trong repository           |
| `git checkout <branch>` | Chuyển đổi giữa các nhánh                             |
| `git merge <branch>`    | Gộp nhánh chỉ định vào nhánh hiện tại                 |
| `git pull`              | Lấy và hợp nhất thay đổi mới nhất từ repository từ xa |
| `git push`              | Đẩy các commit lên repository từ xa                   |
